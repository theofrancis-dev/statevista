About This Website

This website was developed as part of the Front-End Development course at Western Governors University Student: Teofilo Francisco, BSSE
Send Email

The purpose of this project is to demonstrate skills in HTML, CSS, and JavaScript by creating a website that presents information about a state and three of its cities.

The website includes:

  • A responsive design with consistent navigation across all pages.
  • Structured data about each city's population, history, and classification.
  • Implementation of lists, tables, and external links.
  • An interactive form with JavaScript validation.
  • Use of CSS positioning techniques (float, absolute, and relative) to organize content.

This project follows best practices in web development and emphasizes usability and accessibility in user interface design.
